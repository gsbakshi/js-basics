About the Projects

Basic Calculator

This is a basic calculator that will perform basic arithmetic operations. It will accept numbers and operators and will return the result along with logging the calculation.

Monster Killer

This is a monster killer game. It will randomly generate a monster and a player.

  • The player will have to kill the monster by attacking it with 2 kinds of attacks.
  • The player can heal themselves by drinking a potion.
  • The player can also view the battle logs in the console.
